Functions§

  • burn
    The burn function allows a user to remove liquidity from a specified liquidity pool. This is done by burning LP tokens and receiving the respective underlying assets in return.
  • mint
    The mint function allows a user to add liquidity to a liquidity pool. Given two assets and their amounts, it either creates a new liquidity pool if it does not already exist for these two assets or adds the provided liquidity to an existing pool. The user will receive LP tokens in return.
  • swap
    The swap function allows a user to exchange one type of token for another within a specific liquidity pool.
